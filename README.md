# My Little Karaoke theme

This repository is a more up-to-date version of
https://mylittlekaraoke.com/store/Extras/theme.zip
(available from https://www.mylittlekaraoke.com/index.php?pages/Extras),
with the following change:
* All videos have been removed

If you need the videos, download them from the original source, and search for `bg-video` in `themes/MyLittleKaraoke/mlk.ini` and `themes/MyLittleKaraoke/polka.ini`.
You will need to manually adjust some comments.

## Note on artwork
All of the artwork was present in the source mentioned above.
There is no easy way to link this back to their original creators.
No infringement is intended.
